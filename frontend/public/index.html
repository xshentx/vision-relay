视觉模型

图片会先交给这里选中的视觉模型解析，再把解析结果交给文本模型。

视觉模型需要能描述图片内容。不要使用 llama-guard、safety、moderation 这类安全分类模型，否则只会返回 safe/unsafe。

OpenCode 客户端接入

自定义 OpenAI-compatible 模型需要声明图片附件能力，否则 OpenCode 会在本地拦截图片，不会把图片发到中转服务。

Codex 客户端接入

Codex 走 Responses 接口。把 Base URL 指向本机中转后，图片会先交给视觉模型解析，再转给当前文本模型继续完成代码任务。

Claude Code 客户端接入

Claude Code 走 Anthropic Messages 接口。中转会把 Anthropic 请求转换到当前文本模型，并保留工具调用和图片识别结果。
