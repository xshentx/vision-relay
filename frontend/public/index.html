客户端配置

一键配置由程序直接写入本地文件，不再启动终端；开启路由后，切换供应商会自动同步配置。

程序内置

Codex

通过 Responses 接口连接本机中转，并保留工具调用和图片解析结果。

OpenCode

配置 OpenAI-compatible 模型的图片附件能力，避免客户端在本地拦截图片。

Claude

通过 Anthropic Messages 接口连接，并保留工具调用和视觉识别结果。

OpenClaw

写入 OpenAI-compatible 自定义供应商、可用模型与图片输入能力，并切换默认模型。

设置

管理本地 API，以及客户端配置文件和程序位置。

本地 API 接口

设置客户端访问 Vision Relay 的监听地址；关闭后，一键配置与供应商切换会把客户端改为直连当前文本供应商。

功能提示关闭本地服务后视觉模型将不可用；客户端会直连当前文本供应商，只有勾选“支持多模态”的文本模型仍可识别图片。
